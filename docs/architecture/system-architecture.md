## 分层架构

### 前端分层

```
┌─────────────────────────────────────────────────────────┐
│                    View Layer                            │
│  - Views (页面视图)                                      │
│  - Components (组件)                                     │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                 Composition Layer                        │
│  - Composables (组合式函数)                              │
│  - Hooks (逻辑复用)                                      │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                   State Layer                            │
│  - Pinia Stores (状态管理)                               │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                  Service Layer                           │
│  - API Services (API 调用)                               │
│  - WebSocket Services (实时通信)                         │
└─────────────────────────────────────────────────────────┘
```

### 后端分层

```
┌─────────────────────────────────────────────────────────┐
│                Controller Layer                          │
│  - REST Controllers (REST API)                           │
│  - WebSocket Handlers (实时处理)                         │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                  Service Layer                           │
│  - Business Services (业务逻辑)                          │
│  - AI Services (AI 服务)                                 │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                 Repository Layer                         │
│  - JPA Repositories (数据访问)                           │
│  - Custom Repositories (自定义查询)                      │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                   Entity Layer                           │
│  - JPA Entities (实体)                                   │
│  - DTOs (数据传输对象)                                   │
└─────────────────────────────────────────────────────────┘
```

## 安全架构

### 认证流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  医生    │ ──▶ │  HIS     │ ──▶ │  his_    │ ──▶ │  LLM     │
│  登录    │     │  认证    │     │  agent   │     │  服务    │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
                      │                │
                      ▼                ▼
                JWT Token          JWT 验证
```

### 数据安全

| 层面 | 措施 |
|------|------|
| 传输 | HTTPS/TLS 加密 |
| 存储 | AES-GCM 加密 |
| 访问 | RBAC 权限控制 |
| 审计 | 完整操作日志 |

## 扩展性设计

### 水平扩展

- 后端无状态设计，支持多实例部署
- Redis 会话共享
- 数据库读写分离

### 插件化设计

- LLM Provider 可插拔
- 语音识别服务可切换
- HIS 接口适配器模式

## 监控与日志

### 监控指标

- API 响应时间
- LLM 调用延迟
- 语音识别准确率
- 系统资源使用率

### 日志收集

- Spring Boot Actuator
- Prometheus + Grafana
- ELK Stack (可选)
