## 缺陷管理

### 严重程度定义

| 等级 | 说明 |
|------|------|
| **高** | 阻塞功能，无法继续测试 |
| **中** | 功能异常，但有变通方案 |
| **低** | 界面问题，不影响功能 |

### 缺陷处理流程

```
发现缺陷 → 记录缺陷 → 分配修复 → 修复完成 → 回归测试 → 关闭缺陷
```
