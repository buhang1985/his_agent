# UC-001 - 智能问诊

## 基本信息

| 属性 | 值 |
|------|-----|
| **编号** | UC-001 |
| **名称** | 智能问诊 |
| **参与者** | 医生 |
| **优先级** | P0 |
| **状态** | 已批准 |

## 用例描述

医生通过语音录入与患者的对话，系统实时转写并生成结构化病历。

## 前置条件

1. 医生已登录系统
2. 患者信息已从 HIS 加载
3. 麦克风权限已授权
4. 网络连接正常

## 后置条件

1. 问诊记录已保存
2. SOAP 病历已生成
3. 转写文本可查阅
4. 诊断建议已生成

## 基本流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      智能问诊流程                                │
└─────────────────────────────────────────────────────────────────┘

  ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
  │ 1.开始   │ ──▶ │ 2.录音   │ ──▶ │ 3.转写   │ ──▶ │ 4.生成   │
  │ 问诊     │     │ 录入     │     │ 文本     │     │ 病历     │
  └──────────┘     └──────────┘     └──────────┘     └──────────┘
                                                          │
                                                          ▼
  ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
  │ 7.结束   │ ◀── │ 6.保存   │ ◀── │ 5.审核   │ ◀── │ 诊断     │
  │          │     │ 回 HIS   │     │ 修改     │     │ 建议     │
  └──────────┘     └──────────┘     └──────────┘     └──────────┘
```

**步骤说明**:

1. **开始问诊**
   - 医生点击"开始问诊"按钮
   - 系统创建问诊会话
   - 系统加载患者基本信息

2. **录音录入**
   - 医生与患者对话
   - 系统通过麦克风录制音频
   - 实时显示录音状态

3. **转写文本**
   - 系统将音频流发送到语音识别服务
   - 实时显示转写文本
   - 支持医学术语校正

4. **生成病历**
   - 医生点击"结束问诊"
   - 系统调用 LLM 生成 SOAP 病历
   - 系统生成诊断建议

5. **审核修改**
   - 医生查看生成的病历
   - 医生可编辑修改病历内容
   - 医生可调整诊断建议

6. **保存回 HIS**
   - 医生确认病历无误
   - 系统通过 postMessage 发送病历到 HIS
   - HIS 返回保存结果

7. **结束**
   - 显示保存成功提示
   - 问诊状态更新为"已完成"

## 备选流程

### 分支 1: 使用备用语音识别

**触发条件**: Deepgram 服务不可用

**流程**:
1. 系统自动切换到 Whisper 本地识别
2. 提示医生"使用本地语音识别模式"
3. 录音结束后统一转写 (非实时)
4. 继续主流程

### 分支 2: 手动输入

**触发条件**: 麦克风不可用或医生选择手动输入

**流程**:
1. 医生点击"手动输入"
2. 显示文本输入框
3. 医生输入问诊内容
4. 跳至步骤 4 (生成病历)

## 异常流程

| 异常 | 处理方式 |
|------|----------|
| 麦克风权限拒绝 | 显示权限设置指引，提供手动输入选项 |
| 语音识别失败 | 切换到备用方案，提示医生重试 |
| LLM 生成超时 | 显示"生成超时，请重试"，保留转写文本 |
| HIS 保存失败 | 提供导出功能，提示医生手动复制 |
| 网络中断 | 本地缓存数据，网络恢复后同步 |

## 相关接口

### 后端 API

- `POST /api/v1/consultations` - 创建问诊会话
- `POST /api/v1/consultations/{id}/transcribe` - 语音转写
- `POST /api/v1/consultations/{id}/generate-soap` - 生成 SOAP 病历
- `GET /api/v1/consultations/{id}/diagnosis` - 获取诊断建议
- `PUT /api/v1/consultations/{id}/soap-note` - 保存病历

### 前端服务

- `useVoiceConsultation()` - 问诊状态管理 Composable
- `api.createConsultation()` - 创建问诊 API
- `api.transcribeAudio()` - 语音转写 API
- `api.generateSoapNote()` - 病历生成 API

### WebSocket

- `ws://localhost:8080/ws` - 实时转写推送
